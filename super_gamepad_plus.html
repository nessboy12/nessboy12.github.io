Basic Operation Instructions

Hold select to turn the controller on or off.

Hold the following buttons on power up to boot into the following modes:

A -> Nintendo Switch Mode

X -> XInput Mode

B -> DInput Mode

Dpad Left -> NES/SNES Mode

Dpad Down -> N64 Mode

Dpad Right -> GameCube Mode

If no USB cable is connected, the controller will boot into wireless mode.

HOJA EFM8UB1 Update Required

This update can only be run from Windows currently.

After flashing the ESP32 firmware, hold L and R while plugging in.

This will enable USB bootloader on the EFM8UB1. Download the zip file linked

unzip it, and run the included .bat file to update the firmware.

Unplug and replug to finalize the firmware install.
